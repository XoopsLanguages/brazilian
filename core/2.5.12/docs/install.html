Instalando o XOOPS

Instalação pela primeira vez

Prefácio:

XOOPS é um sistema de publicação na web orientado a objetos de código aberto escrito em PHP. É uma ferramenta ideal para desenvolver sites de comunidade dinâmicos de pequeno a grande porte, portais internos de empresas, portais corporativos, blogs e muito mais.

O XOOPS é lançado sob os termos da Licença Pública Geral GNU (GPL) versão 2 ou superior, e é gratuito para uso e modificação. É livre para redistribuição, desde que você respeite os termos de distribuição da GPL.

Requisitos

  • Servidor WWW (Apache, NGINX, IIS, etc)
  • PHP 5.6.0 ou superior, 7.4+ recomendado
  • MySQL 5.6 ou superior, MySQL 8.0+ recomendado

Antes de instalar

  1. Configure corretamente o servidor WWW, PHP e o servidor de banco de dados.
  2. Prepare um banco de dados para o seu site XOOPS.
  3. Crie uma conta de usuário e conceda acesso ao banco de dados para esse usuário.
  4. Torne os diretórios e arquivos graváveis: %s
  5. Por motivos de segurança, é altamente recomendável mover os dois diretórios abaixo para fora do document root e alterar os nomes das pastas: %s
  6. Crie (se ainda não existirem) e torne os diretórios graváveis: %s
  7. Ative cookies e JavaScript no seu navegador.

Instalando localmente

Se você está executando um ambiente local para desenvolvimento ou teste, certifique-se de que os requisitos anteriores foram atendidos. Feito isso, copie o conteúdo do diretório htdocs (do arquivo de distribuição do XOOPS) para o caminho do documento raiz do seu ambiente web. Após copiar os arquivos, você pode iniciar a instalação apontando seu navegador para "http://seu-site.com". Isso iniciará o processo de instalação.

Instalando em uma plataforma hospedada

Se você está em um ambiente hospedado, descompacte os arquivos do XOOPS localmente ou no servidor, se você tiver acesso ao shell (ou seja, SSH). Após isso, certifique-se de mover ou copiar todos os arquivos do XOOPS do diretório htdocs para o diretório web raiz (seu provedor geralmente fornece este local com instruções). Após copiar os arquivos, você pode iniciar a instalação apontando seu navegador para "http://seu-site.com". Isso iniciará o processo de instalação.
