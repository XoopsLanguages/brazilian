Ajuda: Mensagens Privadas

DESCRIÇÃO

O módulo MP é para mensagens privadas entre usuários

INSTALAR/DESINSTALAR

Nenhuma medida especial necessária, siga o processo de instalação padrão – extraia a pasta /pm para o diretório ../modules. Instale o módulo através de Admin -> Módulo do Sistema -> Módulos.

Instruções detalhadas sobre a instalação de módulos estão disponíveis no Manual de Operações do XOOPS

INSTRUÇÕES DE OPERAÇÃO

Para configurar este módulo, você precisa:

  • Você entra na sua Caixa de Entrada diretamente da página inicial ou clicando no link superior “Ir para o módulo” na área de Administração do MP.
  • Configure suas preferências para o módulo (veja “Preferências”) e, opcionalmente, o bloco de MP se você pretende usá-lo (veja “Blocos”).
  • Você pode fazer exclusões em massa de suas mensagens usando a aba “Limpar Mensagens”.
