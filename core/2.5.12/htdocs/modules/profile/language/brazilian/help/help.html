Ajuda: Perfil

DESCRIÇÃO

O módulo Perfil é para gerenciar campos de perfil de usuário personalizados.

INSTALAR/DESINSTALAR

Nenhuma medida especial necessária, siga o processo de instalação padrão – extraia a pasta /profile para o diretório ../modules. Instale o módulo através de Admin -> Módulo do Sistema -> Módulos.

Instruções detalhadas sobre a instalação de módulos estão disponíveis no Manual de Operações do XOOPS

INSTRUÇÕES DE OPERAÇÃO

Para configurar este módulo, você precisa:

  • Configure suas preferências para o módulo (veja ‘Preferências’) e, opcionalmente, o bloco de Perfil se você pretende usá-lo (veja ‘Blocos’).
  • Edite as Categorias existentes ou adicione novas.
  • Edite os Campos existentes ou adicione novos. Aqui você pode especificar quais campos serão visíveis em qual categoria e se serão visíveis durante o registro do usuário.
  • Defina a ordem das etapas de Registro.
  • E, finalmente, você pode definir permissões para campos individuais - quais são editáveis e quais são pesquisáveis.

Medidas Anti-Spam

Para minimizar registros de spam, faça o seguinte:

  • vá para o módulo Protector no Admin, vá para Preferências e, em seguida, na parte inferior, nesta opção: "Stop Forum Spam" Verifica os dados POST em relação a spammers registrados no banco de dados www.stopforumspam.com.
    Defina como "Banir o IP (sem limite)"

  • em /class/captcha/config.php, certifique-se de que o modo esteja definido como "text":

    return $config = array(
    'disabled' => false, // Desativar CAPTCHA
    'mode' => 'text', // modo padrão, você pode escolher 'text', 'image', 'recaptcha'(requer chave api)
    'name' => 'xoopscaptcha', // nome do captcha
    'skipmember' => true, // Ignorar verificação CAPTCHA para membros
    'maxattempts' => 10, // Máximo de tentativas por sessão
    );

  • No módulo Perfil, vá para Admin e na etapa Básica localizada em:
    /modules/profile/admin/step.php?id=1
    defina "Salvar após parar" como "Não"


  • Nas Preferências do Perfil, defina a opção "Usar Captcha após a segunda etapa de registro" como "Sim" (é o padrão)
