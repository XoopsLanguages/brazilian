Ajuda: Enviar E-mail para Usuários

Descrição

O XOOPS possui um sistema de mensagens integrado que permite enviar uma mensagem para toda a base de usuários registrados ou subgrupos deles, ou utilizando filtros de data. Isso pode ser feito na forma de uma mensagem privada (ou seja, através do sistema de mensagens integrado do XOOPS) ou por e-mail. Uma variedade de opções está disponível para filtrar subgrupos de usuários de sua base de membros.

Você também pode usar tags predefinidas que permitirão personalizar sua mensagem.
