Ajuda: Manutenção

Descrição

Manutenção fornece várias funções básicas para ajudá-lo a manter sua instalação XOOPS saudável:

- limpar a pasta de cache
- esvaziar a tabela de sessão
- realizar manutenção das tabelas do banco de dados (analisar, verificar, otimizar e reparar)

Você também pode realizar um "Dump" de todo o banco de dados ou tabelas individuais para fins de backup.
