Ajuda: Módulos

Descrição

Módulos são as construções de software que, aproveitando a infraestrutura fornecida pelo XOOPS, permitem que o webmaster ofereça aos usuários do site conteúdo dinâmico ou estático gerado por meio de uma interface Web que, na maioria das vezes, é muito fácil de usar.

Parte do poder do XOOPS reside na maneira como os módulos são gerenciados. O webmaster pode facilmente instalar, desinstalar, ativar ou desativar qualquer módulo específico (exceto o módulo do Sistema, por razões óbvias). Também é possível renomear módulos e determinar a ordem em que eles aparecerão para o usuário final no menu do site (se você quiser que eles apareçam), porque também é possível fornecer funcionalidades ocultas, definindo módulos ativos que não têm presença visível para o usuário casual. Por exemplo, você poderia definir um módulo como não visível no menu desmarcando o ícone de verificação na coluna Menu. Assim, para o módulo de Notícias, você poderia enviar uma mensagem privada para um grupo selecionado de seus usuários e pedir que acessem esse diretório digitando o endereço no campo de URL, algo como http://www.seusite.com/modules/news.

Para mudar a ordem dos módulos (que será refletida no Menu), basta arrastar e soltar os módulos na posição desejada.

A instalação de módulos requer algum planejamento, tanto para os módulos a serem exibidos quanto para o layout nas páginas. É uma boa ideia revisar outros sites XOOPS e ver quais módulos são usados para exibir vários tipos de conteúdo. Isso pode ser feito facilmente abrindo qualquer site XOOPS e passando o mouse sobre os links do menu nas laterais esquerda e direita. Ao mesmo tempo, observe os URLs exibidos na barra de status do seu navegador. Você deverá ver endereços como este: http://o_site.com/modules/news/.... Todos os módulos são armazenados no diretório /modules/, portanto, o seguinte trecho do endereço (neste exemplo 'news') mostra o módulo que exibirá o conteúdo daquele link do menu.

Se você abrir a página do link, ela mostrará um exemplo de como o módulo é usado e o tipo de conteúdo que exibe. Revisar outros sites dessa forma também fornecerá uma boa visão sobre as capacidades de layout e design.
